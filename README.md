# TabEx
Add Chrome-like tab functionality to explorer, replacing Clover and ExTab.

## 📥 下载

**最新版本**: 从 [Releases](../../releases/latest) 页面下载 `TabExplorer.exe`

无需安装Python环境，下载即用！

---

## ✨ 特色功能

### 核心功能
✅ **完整的Windows资源管理器体验**（每个标签页嵌入真实的explorer.exe窗口，包含完整的导航窗格、地址栏、工具栏、搜索框等所有原生功能）
✅ **自定义标题栏**（无边框窗口设计，可拖动、双击最大化/还原）
✅ **独立书签栏**（位于标签页上方，快速访问常用文件夹）
✅ 多标签页浏览（每个标签页独立路径）
✅ 标签页固定（带 📌 图标，开机自动恢复）
✅ 标签页拖拽排序（鼠标拖拽调整标签页顺序，固定标签保持在最左侧）

### 书签功能
✅ 书签管理（支持文件夹层级、上移/下移、编辑/删除）
✅ 默认书签（🖥️此电脑、🗔桌面、🗑️回收站、🚀启动项、⬇️下载 - 自动维护，不可删除）
✅ 自定义书签（支持📑普通书签和📁文件夹分组）
✅ 书签栏一键访问（点击立即跳转到对应路径）

### 高级功能
✅ 拖拽文件夹打开新标签页
✅ 网络路径支持（UNC 路径 \\\\server\\share）
✅ 特殊 shell 路径支持（回收站、桌面、此电脑等）
✅ 双击标签栏空白区域新建标签页
✅ **单实例模式**（右键菜单打开文件夹时在已有窗口中打开新标签页，避免启动多个进程）
✅ **强大的搜索功能**（支持文件名、文件内容、文件夹名搜索，支持文件类型过滤）
✅ **自动接管资源管理器**（默认开启，类似 Clover，自动将新打开的资源管理器窗口转为 TabExplorer 标签页）
✅ **设置界面**（可配置接管资源管理器功能，设置持久化保存）

## 🎨 界面说明

### 自定义标题栏
窗口顶部的自定义标题栏提供以下功能：
- **🔖 书签管理** - 打开书签管理对话框，添加、编辑、删除、排序书签
- **⚙️ 设置** - 打开设置对话框，配置接管资源管理器等选项
- **— 最小化** - 最小化窗口
- **□/❐ 最大化/还原** - 切换窗口最大化状态
- **✕ 关闭** - 关闭程序

标题栏支持：
- 拖动窗口移动
- 双击最大化/还原
- 在最大化状态下拖动自动还原窗口

### 书签栏
位于标题栏下方、标签页上方的灰色横条，显示所有书签：
- **默认书签**（前5个）：🖥️此电脑、🗔桌面、🗑️回收站、🚀启动项、⬇️下载
  - 自动维护，始终保持在最前面
  - 不会显示在书签管理界面中
  - 无法删除或编辑
- **用户书签**：以📑图标显示
- **书签文件夹**：以📁图标显示，点击展开下拉菜单

## 🚀 完整的Windows资源管理器体验

TabExplorer 最大的特色是将**真正的Windows资源管理器**嵌入到每个标签页中！

**与传统文件管理器的区别**：
- ❌ 传统方案：使用简化的ActiveX控件，功能受限
- ✅ TabExplorer：嵌入完整的explorer.exe窗口，100%原生体验

**每个标签页都包含完整的Explorer功能**：
- 📂 左侧导航窗格（快速访问、OneDrive、此电脑等）
- 🔍 顶部搜索框（实时搜索）
- 📍 地址栏（快速导航、路径复制）
- 🎨 工具栏/Ribbon界面（所有Windows资源管理器功能）
- 👁️ 各种视图模式（大图标、列表、详细信息等）
- 🏷️ Git Tortoise等覆盖图标完美支持

## 🚀 自动接管资源管理器

TabExplorer **默认自动启动**接管功能，像 Clover 一样将系统资源管理器窗口转为标签页！

**工作原理**：
- 自动检测新打开的Explorer窗口
- 直接嵌入到TabExplorer的新标签页中
- 原窗口无缝转换，不丢失任何内容

**控制方式**：
- 点击标题栏 **"⚙️ 设置"** 按钮 → **"接管资源管理器"** 复选框可随时开关
- 设置自动保存到 config.json，重启后保持
- 默认已启用，无需手动操作

**适用场景**：
- 按 `Win + E` 打开资源管理器
- 双击桌面文件夹
- 从开始菜单打开文件夹
- VSCode等软件跳转打开文件夹
- 任何方式打开的Explorer窗口

## 📖 快捷操作

### 标签页操作
- **新建标签页**：点击 ➕ 按钮 或 双击标签栏空白区域
- **关闭标签页**：鼠标悬停在标签页上，点击出现的 × 按钮
- **固定标签页**：右键标签页 → 选择"固定标签页"
- **拖拽排序**：直接拖动标签页调整顺序

### 书签操作
- **访问书签**：点击书签栏上的书签按钮
- **管理书签**：点击标题栏 🔖 按钮打开书签管理器
- **添加书签**：在书签管理器中点击相应按钮
- **书签排序**：在书签管理器中使用"上移"/"下移"按钮

### 窗口操作
- **拖动窗口**：拖动标题栏
- **最大化/还原**：双击标题栏 或 点击 □ 按钮
- **搜索文件**：点击 🔍 按钮
- **刷新当前页**：按 F5 键

## 📝 配置文件

### config.json
保存程序设置：
- `enable_explorer_monitor`: 是否启用接管资源管理器（默认 true）
- `pinned_tabs`: 固定标签页的路径列表

### bookmarks.json
保存书签数据（Chrome书签格式）：
- 默认书签自动维护
- 用户自定义书签和文件夹
- 支持多层级文件夹结构

## 🔧 技术特性

- 基于 PyQt5 开发
- 使用 QAxWidget 嵌入真实 Explorer 窗口
- 无边框窗口 + 自定义标题栏设计
- 单实例模式（Socket通信）
- 资源管理器监控（Win32 API）
- 持久化配置（JSON格式）
